== Description ==
This plugin is designed to send WordPress post data to a specified webhook URL. It does this by scheduling the sending of order data to the webhook when WooCommerce order creation and update events occur. read more about the code in README.md

== Installation ==
This section describes how to install the plugin and get it working.
1. Upload [`bluelena-connect`](link-to-github.zip) to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
